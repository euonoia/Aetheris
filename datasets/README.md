Datasets
========

Organize dataset versions under `datasets/`.

Example
-------

datasets/
  barangay178_v1/
    train/
      images/
      labels/
    valid/
      images/
      labels/
    test/
      images/
      labels/
    data.yaml
    README.md

Each dataset folder must include a `README.md` describing collection date, camera sources, number of images, annotation tool, and any annotation notes.
