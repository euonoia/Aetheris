====================================================

GOAL

====================================================

Build a Dataset Engineering Toolkit that automates the preparation of datasets for YOLOv8 transfer learning.

This toolkit is NOT part of the production backend.

It is a standalone research tool used before training.

Its purpose is to transform raw CCTV or dashcam videos into clean, organized datasets suitable for annotation and training.

DUPLICATE DETECTION

====================================================

Implement duplicate frame detection.

Compare consecutive frames using image similarity.

If two frames exceed a similarity threshold (for example 95%), skip the duplicate.

Allow the threshold to be configurable.

Display:

Remaining Images
