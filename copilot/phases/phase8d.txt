Research Toolkit
----------------

I already have a standalone Dataset Engineering Toolkit built with CustomTkinter.

Current modules include:

- Frame Extractor
- Blur Detector
- Duplicate Detector
- Metadata Generator
- Dataset Organizer
- Quality Inspector

However, these modules currently behave like independent tools.

I want to refactor them into ONE integrated preprocessing pipeline similar to professional computer vision dataset engineering software.

==========================================================
GOAL
==========================================================

The toolkit should operate as ONE workflow instead of multiple disconnected applications.

The user should only interact with ONE dashboard.

Each processing stage automatically passes its output to the next stage.

There should be no separate applications for blur detection, duplicate detection, or quality inspection.

Everything should be orchestrated by a central Pipeline Manager.
